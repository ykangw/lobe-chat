# 图片与视频生成重设计

本周 LobeHub 全面升级了图片与视频生成体验，让创作和浏览视觉内容更加便捷。

## 重要更新

- 图片与视频生成重设计：全新的生成界面，新增图片 / 视频切换功能，轻松在两种创作模式间自由切换
- 记忆管理：支持一键清除所有记忆条目，快速重置对话记忆
- Bot 改进：重构 Bot 内部架构，提升可靠性和可扩展性

## 体验优化

修复压缩视图的显示异常，改进移动端菜单交互，修正消息计数显示的准确性。
